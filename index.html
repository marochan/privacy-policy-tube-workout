プライバシーポリシー / Privacy Policy

「Tubeおうちトレ記録」は、ユーザーの情報を守るため以下の通りポリシーを定めます。

1. 個人情報の収集について

当アプリは、ユーザーの個人情報（名前、住所、メールアドレスなど）を収集しません。また、外部サーバーへデータを送信することもありません。

2. データの保存とバックアップについて

トレーニング記録や登録されたYouTube URLなどのデータは、ユーザーの端末内部（ローカルストレージ）にのみ保存されます。 バックアップ機能を使用する場合、ユーザーの操作により端末内の特定フォルダにファイルが生成されますが、このファイルが開発者や第三者に自動的に送信されることはありません。

3. 第三者サービス（YouTube）の利用について

当アプリはYouTubeの動画を再生するためにYouTube APIサービスを利用しています。そのため、ユーザーはYouTube利用規約（https://www.youtube.com/t/terms）およびGoogleプライバシーポリシー（https://policies.google.com/privacy）に同意したものとみなされます。 YouTubeコンテンツの再生履歴などは、Google社の規定に基づき処理される場合があります。

"Tube Home Workout Log" establishes the following policy to protect user information.

1. Collection of Personal Information

This app does not collect any personal information from users (such as name, address, or email address). Furthermore, no data is transmitted to external servers.

2. Data Storage and Backup

Data such as workout logs and registered YouTube URLs are stored only within the user's device (local storage). When using the backup function, a file is generated in a specific folder on the device by user operation, but this file is never automatically transmitted to the developer or any third party.

3. Use of Third-Party Services (YouTube)

This app uses YouTube API Services to play YouTube videos. Therefore, users are deemed to have agreed to the YouTube Terms of Service (https://www.youtube.com/t/terms) and the Google Privacy Policy (https://policies.google.com/privacy). Playback history and other YouTube content data may be processed in accordance with Google's regulations.
