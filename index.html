プライバシーポリシー

「Tubeおうちトレ記録」は、ユーザーの情報を守るため以下の通りポリシーを定めます。

1. 収集する情報

当アプリはYouTube API Servicesを利用し、動画情報（タイトル、再生時間等）を表示します。また、個人のトレーニング記録を保存しますが、これらはデバイス内にのみ保存され、外部サーバーへ送信されることはありません。

2. YouTube APIの利用

本アプリの利用者は、YouTubeの利用規約（https://www.youtube.com/t/terms）およびGoogleのプライバシーポリシー（http://www.google.com/policies/privacy）に同意したものとみなされます。
